# Paradise Nursery

A React application for Paradise Nursery, featuring plant listings, cart functionality, and more.

## Features

- Landing page with company name and Get Started button
- About page with company information
- Product listing page with multiple categories and plants
- Cart functionality using Redux
- Navigation bar across all pages
- Responsive design

## Technologies Used

- React
- Redux Toolkit
- React Router
- CSS

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the app: `npm start`
